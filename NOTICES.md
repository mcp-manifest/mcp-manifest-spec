## Specification Origin

The mcp-manifest format was originated by **David H Friedel Jr.** in 2026 as part of the ai-manifests protocol family ([mcp-manifest.dev](https://mcp-manifest.dev)). The specification text is released under the [Community Specification License 1.0](LICENSE.md) (Patent and Copyright). The companion specifications — [ADP](https://github.com/ai-manifests/adp-manifest-spec) (deliberation protocol), [ADJ](https://github.com/ai-manifests/adj-manifest-spec) (journal), and [ACB](https://github.com/ai-manifests/acb-manifest-spec) (cognitive budget) — share the same origin and licensing model.

mcp-manifest is a manifest format that wraps installation, configuration, and discovery metadata around servers built on Anthropic's Model Context Protocol (MCP). MCP itself is Anthropic's protocol; the mcp-manifest format is a separate specification that complements it.
